Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element defines the title for a webpage and it very important for Search Engine Optimization.
  • HTML is used to create structure of a webpage

CSS

  • CSS is used to style and layout webpages
  • CSS defines the color, aesthetics, visual layout and organization.
  • Also known as Cascading Style Sheets.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git allows for several people to code at the same time on the same project.
  • Git uses commands known as Git commands.
  • git status: checks what branch you're currently in.

JavaScript

  • JavaScript is a programming language that is on the core technologies of the World Wide Web.
  • JavaScript is a text-based programming language that allows you to make web pages interactive.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
